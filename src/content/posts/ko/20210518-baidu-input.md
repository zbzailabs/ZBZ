macOS용 바이두 입력기 스킨, 미니멀리스트 스타일, 고도로 사용자 정의 가능.

## 스킨 특징

1. Mac용 바이두 입력기에 적합합니다.
2. macOS Monterey에 완벽하게 적응했습니다.
3. 미니멀리스트 스타일, 입력에 집중.
4. 첫 번째 선택을 강조 표시하고 단어 선택에 더 집중하십시오.
5. 고도로 사용자 정의 가능하며 필요에 따라 개인화할 수 있습니다.

## 업데이트 로그

### `V1.1` 2021-11-05

1. 통일된 파일 명명 규칙.
2. 쉬운 사용자 정의를 위해 비고를 추가했습니다.
3. 오른쪽의 페이지 넘기기 아이콘을 제거하여 전체를 더 간단하게 만들었습니다.
4. `global.ini` 파일을 추가했습니다.
5. 흑연 스킨을 추가했습니다.
6. 스킨 예제를 업데이트했습니다.

### `V1.0` 2021-05-18

1. 스킨이 출시되었습니다.

## 사용 지침

1. 특별한 요구 사항이 없는 경우 `examples` 폴더에서 `.bdskk` 형식 스킨 패키지를 직접 다운로드하고 두 번 클릭하여 설치하고 사용할 수 있습니다.
2. 사용자 정의가 필요한 경우 파일을 수정할 수 있습니다.
3. 스킨 리소스 폴더의 내용을 `.zip` 형식 압축 패키지로 압축하고 전체 폴더를 압축하지 마십시오.
4. 압축 패키지를 `.bdskk` 형식 스킨 패키지로 수정합니다.
5. 설치 패키지를 두 번 클릭하여 바이두 입력기로 가져옵니다.

## 파일 구조

```json
└── src
    ├── global.ini
    ├── horizontal.ini
    ├── single.ini
    ├── statusbar.ini
    └── *.png
└── examples
└── README.md
└── LICENSE
```

1. `src`: 리소스 폴더.

2. `globe.ini`: 전역 정의 파일, 스킨 이름, 스킨 유형, 스킨 설명, 작성자 ID, 이메일, 웹 사이트 등을 정의할 수 있습니다. 관련 정보는 바이두 Mac 입력기 모양 설정 페이지에 표시됩니다.

3. `horizontal.ini`: 이중선 모드 구성 파일. 이중선 스킨은 입력 코드와 후보 단어를 동시에 표시하는 모드를 나타냅니다.

   **입력 영역 및 후보 단어 영역**: 그림과 같이 이중선 모드의 배경 이미지 범위를 프레임으로 지정합니다. 두 영역의 각 매개변수 설정 원리는 동일합니다.

   **9칸 영역**: 후보 표시줄 배경은 9칸 확장 방식으로 다양한 후보 단어 길이에 적응합니다. 4개의 매개변수 X / Y / 너비 / 높이는 아래 그림의 녹색 영역을 정의하며, 이는 9칸 그리드의 중간 궁전입니다.

   **콘텐츠 여백**: 입력 코드와 배경 슬라이스 가장자리 사이의 거리입니다. 대략

   **글꼴 및 색상**: 글꼴 및 후보 단어의 색상은 16진수 RGB 형식을 사용합니다. 각 필드가 참조하는 요소는 아래 그림에 나와 있습니다.

   **간격 기호**: 디지털 일련 번호와 후보 단어 사이의 간격 기호를 설정합니다. 공백은 SPACE(대문자)이며 다른 문자는 직접 입력됩니다.

4. `single.ini`: 한 줄 모드 구성 파일. 한 줄 스킨은 후보 단어만 표시하는 모드를 말하며 Mac에서 더 깨끗하고 단순해 보입니다. 한 줄 모드는 후보 단어 영역의 매개변수만 설정하면 됩니다. 매개변수 설정이 적용되는 원리는 이중선 스킨과 동일합니다.

5. `statusbar.ini`: 상태 표시줄 구성 파일. 상태 표시줄의 아이콘은 상태 표시줄 배경 이미지의 어느 곳에나 배열할 수 있으며 가로 및 세로 좌표만 설정하면 됩니다. 모든 좌표 값은 왼쪽 상단 모서리를 기준으로 합니다.

6. `*.png`: 스킨 슬라이스 파일, 해당 파일을 자신의 슬라이스로 바꿀 수 있습니다.

7. `examples`: 예제 스킨, 직접 다운로드하여 사용할 수 있습니다.

8. `README.md`: 프로젝트 readme 파일.

9. `LICENSE`: 이 프로젝트는 [MIT 라이선스](https://github.com/xiyizhou/BaiduIM-Skin/blob/main/LICENSE)를 따릅니다. 소프트웨어 및 소프트웨어 사본을 사용, 복사, 수정, 병합, 게시, 배포, 재라이선스 및 판매할 수 있으며 프로그램의 필요에 따라 라이선스 조건을 적절한 내용으로 수정할 수도 있습니다.

## 기여

질문, 의문 또는 제안 사항이 있으면 언제든지 문의하십시오.

이 스킨 세트에는 여전히 해결해야 할 다음과 같은 문제가 있습니다. 바이두 입력기 프로그램 코드를 이해하지 못하기 때문에 이 개발자는 짧은 시간 내에 구현할 수 없습니다. 구현할 방법이 있다면 기여를 환영합니다.

- [ ] 상태 표시줄과 이중선 스킨을 완벽하게 만듭니다.

- [ ] 후보 영역 글꼴 및 배경색의 수직 중앙 정렬.

- [ ] 후보 단어 디지털 일련 번호의 글리프, 글꼴 크기 및 색상을 사용자 정의하여 Mac 내장 입력기와 유사한 표시 효과를 얻어 후보 단어를 강조 표시할 수 있습니다.

- [ ] 현재 스킨 형식은 `.bdskk`이며 바이두 Mac 입력기 기본 스킨 형식 `.bpsm`으로 전환해야 합니다.

- [ ] 선호하는 단어 배경색을 사용자 정의할 수 있는 스킨과 유사한 iPhone 및 iPad 스킨을 개발합니다.
