오랫동안 Github에 로그인하지 않았는데 오른쪽 하단에 Arctic Code Vault Contributor 배지가 있는 것을 발견했습니다. 알고 보니 2020년에 제가 기여한 프로젝트가 GitHub Archive Program(GitHub 아카이브 프로그램)에 포함되었습니다.

GitHub Archive Program은 미래 세대를 위해 오픈 소스 소프트웨어를 보존하는 것을 목표로 합니다. Github는 세상이 오픈 소스 소프트웨어에 의해 주도되며, 이것이 현대 문명의 보이지 않는 초석이자 전 인류의 공동 재산이라고 믿습니다. Github는 코드 스냅샷을 QR 코드 형태로 수명이 1,000년인 필름에 저장한 다음, 노르웨이 스발바르 제도의 북극 산 깊은 곳에 있는 폐광된 탄광의 GitHub Arctic Code Vault에 필름을 저장합니다.

스발바르 제도는 국제 [스발바르 조약](https://www.loc.gov/law/help/us-treaties/bevans/m-ust000002-0269.pdf)의 보호를 받으며 비무장 지대입니다. 세계 최북단의 마을이며 지구상에서 가장 외지고 지정학적으로 안정된 인간 거주지 중 하나입니다.

AWA(Arctic World Archive)는 노르웨이 국영 광업 회사인 Store Norske Spitsbergen Kulkompani(SNSK)와 장기 디지털 저장소 제공업체인 Piql AS의 공동 이니셔티브입니다. AWA는 아카이브의 영구 보존에 전념합니다. 필름 릴은 스발바르 제도의 외딴 군도에 있는 폐광된 탄광 내의 밀폐된 챔버 안의 강철 벽 컨테이너에 저장됩니다. AWA는 이미 이탈리아, 브라질, 노르웨이, 바티칸 및 기타 여러 국가의 역사 및 문화 데이터를 보존했습니다.

스발바르 제도는 기후 변화의 영향을 받지만 가까운 미래에는 영구 동토층의 가장 바깥쪽 몇 미터만 영향을 받을 수 있으며 창고는 실질적인 피해를 입지 않을 것입니다. 광산은 유명한 국제 종자 저장고(불과 1마일 거리)와 가까워 인류 집단 지식의 장기적이고 안정적인 아카이브로서 스발바르 제도의 위상을 더욱 강화합니다.

Github의 첫 번째 아카이브는 2020년 2월 2일이었으며 그 후 5년마다 코드가 업데이트됩니다. 코드를 더 안전하게 보호하기 위해 Github는 코드 필름을 보들리 도서관, 알렉산드리아 도서관 및 스탠포드 도서관에도 기증했습니다.

천년 전 중국은 송 진종 천희 4년이었습니다. 이해에 범중엄은 31세, 포증은 21세, 구양수는 13세, 적청은 12세, 소순은 11세, 소옹은 9세, 채양은 8세, 주돈이는 3세, 사마광은 1세, 증공은 1세였습니다. 시간은 참으로 깁니다...
