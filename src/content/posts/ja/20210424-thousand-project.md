長い間Githubにログインしていませんでしたが、右下隅にArctic Code Vault Contributorバッジを見つけました。2020年に私が貢献したプロジェクトがGitHub Archive Program（GitHubアーカイブプログラム）に入れられたことが判明しました。

Github Archive Programは、将来の世代のためにオープンソースソフトウェアを保存することを目的としています。Githubは、世界はオープンソースソフトウェアによって推進されており、それは現代文明の目に見えない礎石であり、全人類の共通の財産であると信じています。Githubは、コードのスナップショットをQRコードの形式で1,000年の寿命を持つフィルムに保存し、そのフィルムをノルウェーのスバルバル諸島の北極の山々の奥深くにある廃止された炭鉱にあるGitHub Arctic Code Vaultに保存します。

スバルバル諸島は、国際的な[スバルバル条約](https://www.loc.gov/law/help/us-treaties/bevans/m-ust000002-0269.pdf)によって保護されており、非武装地帯です。世界最北端の町であり、地球上で最も遠隔で地政学的に安定した人間の居住地の1つです。

AWA（Arctic World Archive）は、ノルウェーの国有鉱業会社Store Norske Spitsbergen Kulkompani（SNSK）と長期デジタルストレージプロバイダーPiql ASの共同イニシアチブです。AWAは、アーカイブの永久保存に専念しています。フィルムリールは、スバルバル諸島の遠隔群島にある廃止された炭鉱内の密閉されたチャンバー内の鋼壁コンテナに保管されています。AWAはすでに、イタリア、ブラジル、ノルウェー、バチカン、その他多くの国からの歴史的および文化的データを保存しています。

スバルバル諸島は気候変動の影響を受けていますが、近い将来、永久凍土の最も外側の数メートルだけが影響を受ける可能性があり、倉庫は実質的に害を受けません。鉱山は有名な世界種子貯蔵庫（わずか1マイル離れている）に近く、人間の集合知の長期的に安定したアーカイブとしてのスバルバル諸島の地位をさらに強化しています。

Githubの最初のアーカイブは2020年2月2日で、その後5年ごとにコードが更新されます。コードをより安全に保護するために、Githubはコードフィルムをボドリアン図書館、アレクサンドリア図書館、スタンフォード図書館にも寄贈しました。
