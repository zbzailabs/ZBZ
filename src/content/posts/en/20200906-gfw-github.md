### Weekly Thoughts

Github blocked the 10046 repository, which is a dangerous signal. If one day Github is completely blocked by the GFW, or Github actively blocks Chinese IPs, then the Chinese Internet will suffer a heavy blow. We cannot place our destiny on the mercy of others, we must have our own backup plan. Gitee is a good alternative, but it is still far from Github. We hope that Chinese developers can work together to build our own open source community.

### Weekly Events

Meng Wanzhou was released, but Sino-US relations have not eased. The United States is still suppressing Chinese technology companies, and we must be prepared for a protracted war.
