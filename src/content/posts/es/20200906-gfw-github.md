### Reflexiones de la semana

Github bloqueó el repositorio 10046, lo cual es una señal peligrosa. Si un día Github es bloqueado completamente por el GFW, o Github bloquea activamente las IP chinas, entonces el Internet chino sufrirá un duro golpe. No podemos poner nuestro destino a merced de otros, debemos tener nuestro propio plan de respaldo. Gitee es una buena alternativa, pero aún está lejos de Github. Esperamos que los desarrolladores chinos puedan trabajar juntos para construir nuestra propia comunidad de código abierto.

### Eventos de la semana

Meng Wanzhou fue liberada, pero las relaciones entre China y Estados Unidos no se han relajado. Estados Unidos sigue reprimiendo a las empresas tecnológicas chinas y debemos estar preparados para una guerra prolongada.
