Hacía mucho tiempo que no iniciaba sesión en Github y encontré una insignia de Arctic Code Vault Contributor en la esquina inferior derecha. Resulta que los proyectos en los que contribuí en 2020 se incluyeron en el GitHub Archive Program (Programa de Archivo de GitHub).

El GitHub Archive Program tiene como objetivo preservar el software de código abierto para las generaciones futuras. Github cree que el mundo está impulsado por el software de código abierto, que es la piedra angular invisible de la civilización moderna y propiedad común de toda la humanidad. Github almacena instantáneas de código en forma de códigos QR en una película con una vida útil de 1000 años, y luego almacena la película en el GitHub Arctic Code Vault en una mina de carbón fuera de servicio en lo profundo de las montañas árticas de Svalbard, Noruega.

Svalbard está protegida por el [Tratado de Svalbard](https://www.loc.gov/law/help/us-treaties/bevans/m-ust000002-0269.pdf) internacional y es una zona desmilitarizada. Es la ciudad más septentrional del mundo y uno de los asentamientos humanos más remotos y geopolíticamente estables de la tierra.

AWA (Arctic World Archive) es una iniciativa conjunta entre la empresa minera estatal noruega Store Norske Spitsbergen Kulkompani (SNSK) y el proveedor de almacenamiento digital a largo plazo Piql AS. AWA se dedica a la preservación permanente de archivos. Los carretes de película se almacenan en contenedores de paredes de acero dentro de una cámara sellada dentro de una mina de carbón fuera de servicio en el remoto archipiélago de Svalbard. AWA ya ha preservado datos históricos y culturales de Italia, Brasil, Noruega, el Vaticano y muchos otros países.

Aunque Svalbard se ve afectada por el cambio climático, en el futuro previsible, solo los pocos metros más externos del permafrost pueden verse afectados, y el almacén no sufrirá daños sustanciales. La mina está cerca de la famosa Bóveda Global de Semillas (a solo una milla de distancia), lo que mejora aún más el estatus de Svalbard como un archivo estable a largo plazo del conocimiento colectivo humano.

El primer archivo de Github fue el 2 de febrero de 2020, y el código se actualizará cada cinco años a partir de entonces. Para proteger el código de manera más segura, Github también donó la película del código a la Biblioteca Bodleian, la Biblioteca de Alejandría y la Biblioteca de Stanford.
