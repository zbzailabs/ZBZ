### 一周思考

如果说 Github 是全球最重要的互联网基础设施，可能没有人反驳，它的重要性不仅在于有数以亿计的开源项目，更重要的是其链接的无数开发人员，美帝锁死我技术树之心不死，下一步会否下令 Github 屏蔽我用户访问？

周末证监会说，我国股市总市值70万亿，占GDP 的70%，也就是巴菲特指标为70%， 巴菲特曾将股市总市值/GNP(GNP约等于GDP) 形容为“衡量任何时刻股市估值的唯一最佳指标”。他认为，若两者之间的比值处于70%至80%的区间之内，这时买进股票就会有不错的收益；但如果在这个比值高于100%时买进股票，就等于在玩火。拳击裁判员召集围观群众上场与拳击手互殴？

### 一周事件

上周银行证券策略周报提出消费股估值过高后，本周消费股跌幅居前 , 估值创纪录的调味品和部分准消费公司风险释放是好事。用长期好赛道“粉饰”短期过热风险是有误导性的！消费是长期好赛道不代表消费股可以“脱离地心引力”，不希望消费股成为“无脚鸟（极乐鸟，传说中任性的鸟）”，盛极而衰。（摘自银行证券策略周报）
