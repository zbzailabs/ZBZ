## Google canvas ppt风格

### 1. **极简主义（Minimalist White）**

- **视觉特征**：大量留白、高对比度的黑色字体、纤细的线条。
- **适用场景**：正式的政府汇报或学术演示。
- **色调**：纯白/浅灰背景 + 墨黑色文字 + 10% 的“云合绿”作为点缀色。
- **心理感受**：专业、严谨、客观，强调数据的真实性。

### 2. **玻璃拟态（Glassmorphism）**

- **视觉特征**：半透明毛玻璃效果、柔和的背景渐变、极具深度的阴影。
- **适用场景**：面向年轻创业者、科技博主或在移动端查看。
- **色调**：多色柔和渐变（紫蓝/青绿）+ 磨砂白浮窗。
- **心理感受**：现代感极强、高端、精致。

### 3. **大地生态（Earth & Organic）**

- **视觉特征**：使用大理石纹路或有机纹理、自然的圆角、温暖的色调。
- **适用场景**：强调“乡村振兴”、“绿色有机”和“人文关怀”的主题。
- **色调**：浅沙色/米色背景 + 深苔藓绿文字 + 赤陶色（土色）辅助。
- **心理感受**：亲和、自然、踏实。

### 4. **金融商务（Corporate Finance）**

- **视觉特征**：严格的栅格系统、蓝灰冷色调、精美的矢量图标。
- **适用场景**：面向投资人（VC/PE）的融资路演或年度业绩汇报。
- **色调**：深蓝（Navy Blue）+ 铂金色 + 关键数据使用高亮蓝。
- **心理感受**：稳重、可靠、具有规模感。

## 马斯克思考框架

### 1. '这个问题的物理学原理是什么？'

马斯克将一切还原到客观现实。

```
我的通讯录增长遇到瓶颈。这个问题的物理学原理是什么？
```

### 2. '如果不依赖现有假设，我该如何解决？'

假设是无形的牢笼。

```
我的定价模式基于竞争对手。如果移除所有假设，我该如何解决？'
```

### 3. '这个问题的基本组成部分是什么？'

马斯克将问题分解为原子。

```
我的业务陷入停滞。这个问题的基本组成部分是什么？
```

### 4. '如果成本不存在，最佳解决方案会是什么样？'

约束太早扼杀创造力。

```
我需要重新设计入职流程。如果成本不存在，最佳解决方案会是什么？
```

### 5. '如果必须削减90%，剩下什么？'

残酷的优先级排序。

```
我的产品路线图有50个项目。如果必须削减90%，剩下什么？
```

### 6. '如果完全失败，根本原因是什么？'

马斯克从失败开始设计成功。

```
我正在推出新产品。如果失败，根本原因是什么？
```

### 7. '如果忽视行业规范，解决方案会是什么？'

马斯克通过拒绝模仿绕过整个行业。

```
我要开始咨询业务。如果忽视行业规范，解决方案会是什么？
```

### 8. '哪部分真的不可能，哪部分只是感觉不可能？'

大多数限制源于情感。

```
我在考虑转型。哪部分感觉不可能而非真的不可能？
```

### 9. '最小可行突破是什么？'

不是最小可行产品。是最小可行突破。

```
我想发展YouTube频道。最小可行突破是什么？
```

### 10. '如果今天重新开始，知道现在的情况，我会构建什么？'

重新审视思考。

```
我有一个随机发展的混乱SaaS。如果今天重新开始，我会构建什么？
```

### 11. '我没有质疑的隐藏约束是什么？'

大多数问题隐藏着虚假的墙。

```
我的代理商停留在20个客户。我没有质疑的隐藏约束是什么？
```

### 12. '如果只关注物理学而非政治，我会如何解决？'

马斯克从问题解决中移除社会摩擦。

```
我需要重组团队。如果只关注物理学，我会如何解决？
```

### 13. '如果必须在10倍速完成，我会怎么做？'

```
我计划6个月发布。如果必须在10倍速完成，我会怎么做？
```

### 14. '如果必须扩展到数百万，会是什么样？'

马斯克以数量级思考。

```
我正在创建小型在线课程。如果必须扩展到数百万，会是什么样？
```
