许久没有登录 Github，发现右下角多了北极库勋章（Arctic Code Vault Contributor） 原来2020年参与贡献的项目被放进了GitHub Archive Program（GitHub 存档程序）。

Github Archive Program 旨在为子孙后代保留开源软件。Github 认为，世界由开源软件驱动，它是现代文明的隐形基石，也是全人类的共同财产。Github 以二维码的形式将代码快照存放在寿命长达1000 年的胶片中，然后把胶片存储在挪威斯瓦尔巴特群岛北极山深处退役的煤矿中的GitHub北极代码保管库中。

斯瓦尔巴群岛受国际《[斯瓦尔巴群岛条约》](https://www.loc.gov/law/help/us-treaties/bevans/m-ust000002-0269.pdf)的保护，属于非军事区。它是世界上最北端的城镇，是地球上最偏远和地缘政治最稳定的人类居住地之一。

AWA（北极世界档案馆）是挪威国有采矿公司Store Norske Spitsbergen Kulkompani（SNSK）与长期数字存储提供商Piql AS的一项联合计划。AWA致力于永久保存档案。胶片卷轴存储在斯瓦尔巴特群岛偏远群岛的一座退役煤矿内的密闭室内的钢壁容器中。AWA已经保存了来自意大利，巴西，挪威，梵蒂冈和许多其他国家的历史和文化数据。

尽管斯瓦尔巴群岛受到气候变化的影响，但在可预见的将来，可能仅有最外层的几米冻土受影响，仓库不会受到实质性危害。矿山靠近著名的全球种子库（仅一英里之遥），这进一步增强了斯瓦尔巴群岛作为长期稳定的人类集体知识档案馆的地位。

Github 的首次存档时间是2020年2月2日，以后每五年会更新一次代码。为了更安全的保护代码，Github 还将代码胶片捐赠给了 Bodleian图书馆，亚历山大图书馆和斯坦福图书馆。
