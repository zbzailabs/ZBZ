Saya tiba di Bandara Huanghua jam 8 pagi. Saya berencana untuk langsung pergi ke Museum Hunan, tetapi saya melihat maglev di aula kedatangan, jadi saya sementara memutuskan untuk naik maglev ke stasiun kereta api berkecepatan tinggi dan kemudian transfer ke kereta bawah tanah ke museum.

Meskipun Maglev Shanghai sudah tua, ia memiliki tampilan kecepatan. Maglev Changsha tampaknya dibangun lebih lambat dari Shanghai, tetapi tidak ada prompt kecepatan di gerbong, dan kecepatannya tidak melonjak, membuatnya tidak berbeda dari kereta bawah tanah. Namun, harganya hanya setengah dari Shanghai.

Ada koridor pendidikan merah panjang dan pengadilan militer di Jalan Yingbin, dan kemudian ada Taman Martir. Di seberang taman adalah museum. Sayangnya, saya ceroboh dan tidak membuat janji sehari sebelumnya, jadi saya tidak bisa masuk museum. Saya kembali ke Taman Martir dengan kesal, berjalan mengelilingi menara peringatan tiga kali, dan menyaksikan sekelompok bibi menari.

Saat makan malam, saya memberi tahu teman saya bahwa sulit melihat anak muda di kereta bawah tanah dan di jalan, dan tidak banyak mobil kelas menengah ke atas di jalan. Saya merasa Changsha kurang vitalitas. Teman saya mengatakan bahwa alasan utamanya adalah harga rumah di Changsha tidak naik selama 10 tahun. Saya juga berpikir begitu. Jika yang lain naik dan Anda tidak, atribut investasi properti hilang, dan tidak ada yang akan datang untuk tinggal.
