لم أقم بتسجيل الدخول إلى Github لفترة طويلة، ووجدت شارة Arctic Code Vault Contributor في الزاوية اليمنى السفلية. اتضح أن المشاريع التي ساهمت فيها في عام 2020 تم وضعها في برنامج أرشيف Github (GitHub Archive Program).

يهدف برنامج أرشيف Github إلى الحفاظ على البرمجيات مفتوحة المصدر للأجيال القادمة. تعتقد Github أن العالم مدفوع بالبرمجيات مفتوحة المصدر، وهي حجر الزاوية غير المرئي للحضارة الحديثة والملكية المشتركة للبشرية جمعاء. تقوم Github بتخزين لقطات التعليمات البرمجية في شكل رموز استجابة سريعة (QR codes) على فيلم بعمر افتراضي يصل إلى 1000 عام، ثم تقوم بتخزين الفيلم في قبو Github للكود القطبي الشمالي في منجم فحم تم إيقاف تشغيله في أعماق جبال القطب الشمالي في سفالبارد، النرويج.

تتمتع سفالبارد بحماية [معاهدة سفالبارد](https://www.loc.gov/law/help/us-treaties/bevans/m-ust000002-0269.pdf) الدولية وهي منطقة منزوعة السلاح. إنها المدينة الواقعة في أقصى الشمال في العالم وواحدة من أكثر المستوطنات البشرية نائية واستقرارًا جيوسياسيًا على وجه الأرض.

أرشيف العالم القطبي الشمالي (AWA) هو مبادرة مشتركة بين شركة التعدين النرويجية المملوكة للدولة Store Norske Spitsbergen Kulkompani (SNSK) وموفر التخزين الرقمي طويل الأجل Piql AS. يكرس AWA للحفاظ الدائم على الأرشيفات. يتم تخزين بكرات الأفلام في حاويات ذات جدران فولاذية داخل غرفة محكمة الغلق داخل منجم فحم تم إيقاف تشغيله في أرخبيل سفالبارد النائي. قام AWA بالفعل بحفظ البيانات التاريخية والثقافية من إيطاليا والبرازيل والنرويج والفاتيكان والعديد من البلدان الأخرى.

على الرغم من تأثر سفالبارد بتغير المناخ، ففي المستقبل المنظور، قد تتأثر فقط الأمتار القليلة الخارجية من التربة الصقيعية، ولن يتضرر المستودع بشكل كبير. يقع المنجم بالقرب من قبو البذور العالمي الشهير (على بعد ميل واحد فقط)، مما يعزز مكانة سفالبارد كأرشيف مستقر طويل الأجل للمعرفة الجماعية البشرية.

كان أول أرشيف لـ Github في 2 فبراير 2020، وسيتم تحديث الكود كل خمس سنوات بعد ذلك. من أجل حماية الكود بشكل أكثر أمانًا، تبرعت Github أيضًا بفيلم الكود لمكتبة بودليان ومكتبة الإسكندرية ومكتبة ستانفورد.
