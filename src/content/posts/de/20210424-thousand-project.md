Ich habe mich lange nicht mehr bei Github angemeldet und unten rechts ein Arctic Code Vault Contributor-Abzeichen gefunden. Es stellt sich heraus, dass die Projekte, zu denen ich 2020 beigetragen habe, in das GitHub Archive Program (GitHub-Archivierungsprogramm) aufgenommen wurden.

Das Github Archive Program zielt darauf ab, Open-Source-Software für zukünftige Generationen zu bewahren. Github glaubt, dass die Welt von Open-Source-Software angetrieben wird, die der unsichtbare Eckpfeiler der modernen Zivilisation und das gemeinsame Eigentum der gesamten Menschheit ist. Github speichert Code-Schnappschüsse in Form von QR-Codes auf Film mit einer Lebensdauer von 1.000 Jahren und lagert den Film dann im GitHub Arctic Code Vault in einem stillgelegten Kohlebergwerk tief in den arktischen Bergen von Spitzbergen, Norwegen.

Spitzbergen ist durch den internationalen [Spitzbergen-Vertrag](https://www.loc.gov/law/help/us-treaties/bevans/m-ust000002-0269.pdf) geschützt und eine entmilitarisierte Zone. Es ist die nördlichste Stadt der Welt und eine der abgelegensten und geopolitisch stabilsten menschlichen Siedlungen der Erde.

AWA (Arctic World Archive) ist eine gemeinsame Initiative des norwegischen staatlichen Bergbauunternehmens Store Norske Spitsbergen Kulkompani (SNSK) und des Anbieters für langfristige digitale Speicherung Piql AS. AWA widmet sich der dauerhaften Aufbewahrung von Archiven. Die Filmrollen werden in stahlwandigen Containern in einer versiegelten Kammer in einem stillgelegten Kohlebergwerk auf dem abgelegenen Archipel von Spitzbergen gelagert. AWA hat bereits historische und kulturelle Daten aus Italien, Brasilien, Norwegen, dem Vatikan und vielen anderen Ländern bewahrt.

Obwohl Spitzbergen vom Klimawandel betroffen ist, sind in absehbarer Zeit möglicherweise nur die äußersten paar Meter des Permafrosts betroffen, und das Lager wird nicht wesentlich geschädigt. Das Bergwerk liegt in der Nähe des berühmten Global Seed Vault (nur eine Meile entfernt), was den Status von Spitzbergen als langfristig stabiles Archiv des kollektiven menschlichen Wissens weiter stärkt.

Das erste Archiv von Github war am 2. Februar 2020, und der Code wird danach alle fünf Jahre aktualisiert. Um den Code sicherer zu schützen, spendete Github den Codefilm auch an die Bodleian Library, die Library of Alexandria und die Stanford Library.
