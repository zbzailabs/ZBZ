### Gedanken der Woche

Github hat das Repository 10046 blockiert, was ein gefährliches Signal ist. Wenn Github eines Tages vollständig von der GFW blockiert wird oder Github chinesische IPs aktiv blockiert, wird das chinesische Internet einen schweren Schlag erleiden. Wir können unser Schicksal nicht der Gnade anderer überlassen, wir müssen unseren eigenen Backup-Plan haben. Gitee ist eine gute Alternative, aber es ist noch weit von Github entfernt. Wir hoffen, dass chinesische Entwickler zusammenarbeiten können, um unsere eigene Open-Source-Community aufzubauen.

### Ereignisse der Woche

Meng Wanzhou wurde freigelassen, aber die Beziehungen zwischen China und den USA haben sich nicht entspannt. Die Vereinigten Staaten unterdrücken weiterhin chinesische Technologieunternehmen, und wir müssen auf einen langwierigen Krieg vorbereitet sein.
