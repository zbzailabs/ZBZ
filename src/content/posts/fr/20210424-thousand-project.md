Je ne me suis pas connecté à Github depuis longtemps et j'ai trouvé un badge Arctic Code Vault Contributor dans le coin inférieur droit. Il s'avère que les projets auxquels j'ai contribué en 2020 ont été intégrés au GitHub Archive Program (Programme d'archivage GitHub).

Le GitHub Archive Program vise à préserver les logiciels open source pour les générations futures. Github estime que le monde est dirigé par des logiciels open source, qui sont la pierre angulaire invisible de la civilisation moderne et la propriété commune de toute l'humanité. Github stocke des instantanés de code sous forme de codes QR sur un film d'une durée de vie de 1 000 ans, puis stocke le film dans le GitHub Arctic Code Vault dans une mine de charbon désaffectée au fond des montagnes arctiques du Svalbard, en Norvège.

Le Svalbard est protégé par le [traité international du Svalbard](https://www.loc.gov/law/help/us-treaties/bevans/m-ust000002-0269.pdf) et est une zone démilitarisée. C'est la ville la plus septentrionale du monde et l'un des établissements humains les plus isolés et géopolitiquement stables de la planète.

AWA (Arctic World Archive) est une initiative conjointe entre la société minière publique norvégienne Store Norske Spitsbergen Kulkompani (SNSK) et le fournisseur de stockage numérique à long terme Piql AS. AWA se consacre à la préservation permanente des archives. Les bobines de film sont stockées dans des conteneurs à parois d'acier à l'intérieur d'une chambre scellée dans une mine de charbon désaffectée dans l'archipel isolé du Svalbard. AWA a déjà préservé des données historiques et culturelles d'Italie, du Brésil, de Norvège, du Vatican et de nombreux autres pays.

Bien que le Svalbard soit affecté par le changement climatique, dans un avenir prévisible, seuls les quelques mètres les plus extérieurs du pergélisol pourraient être affectés, et l'entrepôt ne sera pas substantiellement endommagé. La mine est proche de la célèbre réserve mondiale de semences (à seulement un mile), ce qui renforce encore le statut du Svalbard en tant qu'archive stable à long terme des connaissances collectives humaines.

Le premier archivage de Github a eu lieu le 2 février 2020, et le code sera mis à jour tous les cinq ans par la suite. Afin de protéger le code de manière plus sûre, Github a également fait don du film de code à la bibliothèque Bodleian, à la bibliothèque d'Alexandrie et à la bibliothèque de Stanford.
