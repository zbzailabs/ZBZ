### Réflexions de la semaine

Github a bloqué le dépôt 10046, ce qui est un signal dangereux. Si un jour Github est complètement bloqué par le GFW, ou si Github bloque activement les IP chinoises, alors l'Internet chinois subira un coup dur. Nous ne pouvons pas placer notre destin à la merci des autres, nous devons avoir notre propre plan de secours. Gitee est une bonne alternative, mais elle est encore loin de Github. Nous espérons que les développeurs chinois pourront travailler ensemble pour construire notre propre communauté open source.

### Événements de la semaine

Meng Wanzhou a été libérée, mais les relations sino-américaines ne se sont pas apaisées. Les États-Unis continuent de réprimer les entreprises technologiques chinoises, et nous devons nous préparer à une guerre prolongée.
