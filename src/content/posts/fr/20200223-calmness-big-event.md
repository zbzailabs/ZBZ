**Événements**

La pneumonie à nouveau coronavirus s'est propagée au monde entier. Selon les statistiques de JP Morgan, la situation épidémique mondiale est encore compliquée. JP Morgan divise les pays comptant plus de 250 cas confirmés en quatre étapes :

1. Développement précoce : Les pays signalent de nouvelles infections, mais le nombre total d'infections est relativement faible. Les Pays-Bas (265) / le Royaume-Uni (273) entrent dans cette catégorie.

2. Accélération : Le taux de croissance des nouvelles infections quotidiennes est à la hausse et l'échelle globale des infections est relativement importante. Le nombre d'infections augmente rapidement, mais le taux de guérison est très limité. Les États-Unis (537) / le Japon (502) / l'Espagne (673) / l'Allemagne (1040) / la France (1126) entrent dans cette catégorie.

3. Accumulation tardive : Le taux de croissance des nouvelles infections quotidiennes diminue. Mais les nouveaux cas quotidiens sont encore considérables, mais le nombre de guérisons a un taux de croissance plus élevé. La Corée du Sud (7382) / l'Italie (7375) / l'Iran (6566) entrent dans cette catégorie.

4. Rétablissement : Les nouveaux cas signalés quotidiennement sont limités. L'augmentation du nombre de guérisons est plus importante. La Chine (80735 personnes) est actuellement en phase de rétablissement.
