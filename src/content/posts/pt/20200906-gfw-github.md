### Pensamentos da Semana

O Github bloqueou o repositório 10046, o que é um sinal perigoso. Se um dia o Github for completamente bloqueado pelo GFW, ou o Github bloquear ativamente IPs chineses, a internet chinesa sofrerá um duro golpe. Não podemos colocar nosso destino à mercê dos outros, devemos ter nosso próprio plano de backup. O Gitee é uma boa alternativa, mas ainda está longe do Github. Esperamos que os desenvolvedores chineses possam trabalhar juntos para construir nossa própria comunidade de código aberto.

### Eventos da Semana

Meng Wanzhou foi libertada, mas as relações sino-americanas não diminuíram. Os Estados Unidos ainda estão reprimindo as empresas de tecnologia chinesas, e devemos estar preparados para uma guerra prolongada.
