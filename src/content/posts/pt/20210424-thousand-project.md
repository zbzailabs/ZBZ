Não faço login no Github há muito tempo e encontrei um emblema de Arctic Code Vault Contributor no canto inferior direito. Acontece que os projetos com os quais contribuí em 2020 foram colocados no GitHub Archive Program (Programa de Arquivo do GitHub).

O GitHub Archive Program visa preservar o software de código aberto para as gerações futuras. O Github acredita que o mundo é impulsionado por software de código aberto, que é a pedra angular invisível da civilização moderna e propriedade comum de toda a humanidade. O Github armazena instantâneos de código na forma de códigos QR em filme com uma vida útil de 1.000 anos e, em seguida, armazena o filme no GitHub Arctic Code Vault em uma mina de carvão desativada nas profundezas das montanhas árticas de Svalbard, Noruega.

Svalbard é protegida pelo [Tratado de Svalbard](https://www.loc.gov/law/help/us-treaties/bevans/m-ust000002-0269.pdf) internacional e é uma zona desmilitarizada. É a cidade mais setentrional do mundo e um dos assentamentos humanos mais remotos e geopoliticamente estáveis da Terra.

AWA (Arctic World Archive) é uma iniciativa conjunta entre a empresa de mineração estatal norueguesa Store Norske Spitsbergen Kulkompani (SNSK) e o provedor de armazenamento digital de longo prazo Piql AS. A AWA dedica-se à preservação permanente de arquivos. Os rolos de filme são armazenados em recipientes com paredes de aço dentro de uma câmara selada dentro de uma mina de carvão desativada no remoto arquipélago de Svalbard. A AWA já preservou dados históricos e culturais da Itália, Brasil, Noruega, Vaticano e muitos outros países.

Embora Svalbard seja afetada pelas mudanças climáticas, no futuro previsível, apenas os poucos metros mais externos do permafrost podem ser afetados, e o armazém não será substancialmente prejudicado. A mina fica perto do famoso Global Seed Vault (a apenas um quilômetro e meio de distância), o que aumenta ainda mais o status de Svalbard como um arquivo estável de longo prazo do conhecimento coletivo humano.

O primeiro arquivo do Github foi em 2 de fevereiro de 2020, e o código será atualizado a cada cinco anos depois disso. Para proteger o código com mais segurança, o Github também doou o filme do código para a Biblioteca Bodleian, a Biblioteca de Alexandria e a Biblioteca de Stanford.
