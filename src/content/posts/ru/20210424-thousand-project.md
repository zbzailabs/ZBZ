Я давно не заходил на Github и обнаружил значок Arctic Code Vault Contributor в правом нижнем углу. Оказывается, проекты, в которые я внес свой вклад в 2020 году, были включены в GitHub Archive Program (Программу архивирования GitHub).

GitHub Archive Program направлена на сохранение программного обеспечения с открытым исходным кодом для будущих поколений. Github считает, что миром движет программное обеспечение с открытым исходным кодом, которое является невидимым краеугольным камнем современной цивилизации и общим достоянием всего человечества. Github хранит снимки кода в виде QR-кодов на пленке со сроком службы 1000 лет, а затем хранит пленку в GitHub Arctic Code Vault в выведенной из эксплуатации угольной шахте глубоко в арктических горах Шпицбергена, Норвегия.

Шпицберген находится под защитой международного [Договора о Шпицбергене](https://www.loc.gov/law/help/us-treaties/bevans/m-ust000002-0269.pdf) и является демилитаризованной зоной. Это самый северный город в мире и одно из самых отдаленных и геополитически стабильных человеческих поселений на земле.

AWA (Arctic World Archive) — это совместная инициатива норвежской государственной горнодобывающей компании Store Norske Spitsbergen Kulkompani (SNSK) и поставщика долгосрочного цифрового хранения Piql AS. AWA занимается постоянным сохранением архивов. Катушки с пленкой хранятся в контейнерах со стальными стенами внутри герметичной камеры в выведенной из эксплуатации угольной шахте на отдаленном архипелаге Шпицберген. AWA уже сохранила исторические и культурные данные из Италии, Бразилии, Норвегии, Ватикана и многих других стран.

Хотя Шпицберген подвержен влиянию изменения климата, в обозримом будущем могут пострадать только самые внешние несколько метров вечной мерзлоты, и склад не пострадает существенно. Шахта находится недалеко от знаменитого Всемирного семенохранилища (всего в миле), что еще больше укрепляет статус Шпицбергена как долгосрочного стабильного архива коллективных знаний человечества.

Первый архив Github был создан 2 февраля 2020 года, и после этого код будет обновляться каждые пять лет. Чтобы более надежно защитить код, Github также пожертвовал пленку с кодом Бодлианской библиотеке, Александрийской библиотеке и Стэнфордской библиотеке.

Тысячу лет назад в Китае был четвертый год правления Тяньси Сун Чжэньцзуна. В этом году Фань Чжунъяню был 31 год, Бао Чжэну — 21 год, Оуян Сю — 13 лет, Ди Цину — 12 лет, Су Сюню — 11 лет, Шао Юну — 9 лет, Цай Сяну — 8 лет, Чжоу Дуньи — 3 года, Сыма Гуану — 1 год, а Цзэн Гуну — 1 год.
