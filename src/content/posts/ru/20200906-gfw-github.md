### Мысли недели

Github заблокировал репозиторий 10046, что является опасным сигналом. Если однажды Github будет полностью заблокирован GFW, или Github активно заблокирует китайские IP-адреса, то китайский интернет понесет тяжелый удар. Мы не можем полагаться на милость других, у нас должен быть свой запасной план. Gitee — хорошая альтернатива, но она все еще далека от Github. Мы надеемся, что китайские разработчики смогут работать вместе, чтобы создать собственное сообщество с открытым исходным кодом.

### События недели
