Скин метода ввода Baidu для macOS, минималистичный стиль, широкие возможности настройки.

## Особенности скина

1. Подходит для метода ввода Baidu для Mac.
2. Идеально адаптирован к macOS Monterey.
3. Минималистичный стиль, фокус на вводе.
4. Выделите первый выбор, больше внимания уделяйте выбору слов.
5. Широкие возможности настройки, можно персонализировать в соответствии с потребностями.

## Инструкции по использованию

1. Если нет особых потребностей, вы можете напрямую загрузить пакет скинов в формате `.bdskk` в папку `examples` и дважды щелкнуть, чтобы установить и использовать.
2. Если требуется настройка, файл можно изменить.
3. Сожмите содержимое папки ресурсов скина в сжатый пакет формата `.zip`, не сжимайте всю папку.
4. Измените сжатый пакет на пакет скинов формата `.bdskk`.
5. Дважды щелкните установочный пакет, чтобы импортировать его в метод ввода Baidu.

## Структура файла

```json
└── src
    ├── global.ini
    ├── horizontal.ini
    ├── single.ini
    ├── statusbar.ini
    └── *.png
└── examples
└── README.md
└── LICENSE
```

1. `src`: Папка ресурсов.

2. `globe.ini`: Файл глобального определения, может определять имя скина, тип скина, описание скина, идентификатор автора, электронную почту, веб-сайт и т. д. Соответствующая информация будет отображаться на странице настроек внешнего вида метода ввода Baidu Mac.

3. `horizontal.ini`: Файл конфигурации двухстрочного режима. Двухстрочный скин относится к режиму, который отображает коды ввода и слова-кандидаты одновременно.

   **Область ввода и область слов-кандидатов**: Ограничьте диапазон фонового изображения двухстрочного режима, как показано на рисунке. Принцип настройки каждого параметра в двух областях одинаков.

   **Область девяти квадратов**: Фон панели кандидатов адаптируется к разной длине слов-кандидатов в манере расширения девяти квадратов. Четыре параметра X / Y / Ширина / Высота определяют зеленую область на рисунке ниже, которая является средним дворцом сетки из девяти квадратов.

   **Поле содержимого**: Расстояние между кодом ввода и краем фонового среза. Примерно

   **Шрифт и цвет**: Цвет шрифта и слов-кандидатов использует шестнадцатеричный формат RGB. Элементы, на которые ссылается каждое поле, показаны на рисунке ниже:

   **Символ интервала**: Установите символ интервала между цифровым серийным номером и словом-кандидатом. Пробел — это ПРОБЕЛ (заглавные буквы), а другие символы вводятся напрямую.

4. `single.ini`: Файл конфигурации однострочного режима. Однострочный скин относится к режиму, который отображает только слова-кандидаты, что выглядит чище и проще на Mac. Однострочный режим требует только настройки параметров области слов-кандидатов. Принцип вступления в силу настройки параметров такой же, как и у двухстрочного скина.

5. `statusbar.ini`: Файл конфигурации строки состояния. Значки в строке состояния можно расположить в любом месте фонового изображения строки состояния, вам нужно только установить его горизонтальные и вертикальные координаты. Все значения координат основаны на верхнем левом углу.

6. `*.png`: Файл среза скина, вы можете заменить соответствующий файл своим собственным срезом.

7. `examples`: Примеры скинов, можно загрузить и использовать напрямую.

8. `README.md`: Файл readme проекта.

9. `LICENSE`: Этот проект следует [лицензии MIT](https://github.com/xiyizhou/BaiduIM-Skin/blob/main/LICENSE). Вы можете использовать, копировать, изменять, объединять, публиковать, распространять, сублицензировать и продавать программное обеспечение и копии программного обеспечения, а также вы можете изменять условия лицензии на соответствующее содержимое в соответствии с потребностями программы.
