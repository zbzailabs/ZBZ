# ZBZ Content

ZBZ is a static multilingual editorial site covering startup, investing, and life notes.

## Entry Points

- Read the concise site index at `/llms.txt`.
- Read the complete public post index at `/llms-full.txt`.
- Discover crawlable HTML URLs through `/sitemap-index.xml`.
- Use locale-prefixed routes. The default locale is English at `/en/`.

## Content Rules

- Public content is readable without authentication.
- The site does not expose transactional tools, private APIs, or account actions.
- Respect `robots.txt` and the published Content-Signal directives.
